# PASCI Application Web

## À propos du projet
Plateforme digitale des OSC membres du CRASC.

## Technologies utilisées
- [text](https://nextjs.org/)

## Lancer le projet
- Cloner le projet
```bash
git clone https://github.com/"username"/"repo_name".git
```

- Installer les dépendances (packages):
```bash
cd "project_name"
npm install
```

- Lancer le projet:
```bash
npm run dev
```

Ouvrir [http://localhost:3000](http://localhost:3000) avec votre navigateur pour voir le résultat.
